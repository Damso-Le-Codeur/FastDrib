Comment ça marche ?

1. Préparation des fichiers

Préparez vos fichiers PDF et un fichier CSV avec la liste des destinataires.

2. Importation

Importez vos fichiers via ce formulaire. Le système fera automatiquement la correspondance.

3. Distribution

Envoyez les liens de téléchargement sécurisés par email à chaque destinataire.

4. Suivi

Suivez les téléchargements en temps réel depuis le tableau de bord.

Conseils pratiques

Vérifiez les correspondances

Assurez-vous que chaque code du CSV a un fichier PDF correspondant.

Testez avec un petit groupe

Commencez par un petit groupe pour tester le système avant d'envoyer en masse.

Sécurité garantie

Chaque destinataire ne peut accéder qu'à son propre fichier avec un code unique.
